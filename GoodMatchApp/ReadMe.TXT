Welcome to the Good Match Application

To get started, Open the .sln file using Visual  Studio.
Run the program, This application is a console application

You have 3 options:
OPT 1: ENTER TWO NAMES AND GET THEIR MATCH PERCENTAGE.
This will calculate the match percentage between two peoples first names that you enter.

OPT 2: ENTER THE CSV FILENAME AND MATCH MANY NAMES.
This will read peoples first names from a csv file that you choose and calcute their match percentage

OPT 3: EXIT.
This will exit the application

Enjoy and Have fun.
